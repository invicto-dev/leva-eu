# 🤖 Protocolo do Agente

> **Atuação:** Siga este guia para máxima eficiência e precisão.

## 📌 Gatilhos de Contexto (Ordem de Leitura)
1.  **`MEMORY.md`**: O que mudou e o que falta. (SEMPRE)
2.  **`PROJECT.md`**: Stack e infra.
3.  **`ARCHITECTURE.md`**: Onde as coisas estão.
4.  **`CONVENTIONS.md`**: Como escrever.

## 🛠️ Regras de Ouro (Inflexíveis)
- **Idioma:** Explicações e planos em **PT-BR**. Código em **Inglês**.
- **Edição:** Priorize `replace_file_content` sobre `write_to_file`.
- **Veracidade:** Não invente. Peça clareza se houver dúvida.
- **Persistência:** Atualize `MEMORY.md` após cada entrega.

## 🎯 Modos
- **Pesquisa:** Ver `ARCHITECTURE.md`.
- **Execução:** Ver `CONVENTIONS.md`.
- **Status:** Ver `MEMORY.md`.
